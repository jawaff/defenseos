# DefenseOS

## Prerequisites

- Java 21

## Running

### Using Gradle

```
.\gradlew.bat :defenseos-sawtooth:run
```

### Using UberJar

```
.\gradlew.bat uberJar

java -jar defenseos-sawtooth\build\libs\defenseos-sawtooth-1.0-SNAPSHOT-uber.jar
```

## Generating Javadocs

```
.\gradlew.bat javadoc
```
